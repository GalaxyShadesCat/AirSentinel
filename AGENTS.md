# AGENTS Instructions

- Use British English spelling and phrasing in written documentation and user-facing copy.
- Assume the reader is a layperson (not a domain expert): explain clearly, define technical terms briefly, and avoid unnecessary jargon while preserving technical accuracy.
- Always run tests before finalizing changes.
- Minimum required checks:
  - `npm test`
  - `npm run build`
- If tests fail, fix them before submitting.
